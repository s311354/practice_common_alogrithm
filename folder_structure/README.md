## Q1. Folder structure
+------------------+------+
| Column Name | Type |
+------------------+------+
| id | int |
| p_id | int |
+------------------+------+
id is the primary key column for this table.
Each row of this table contains information about the id of a folder and the p_id is id of
its parent folder.
The given structure is always a valid folder structure.
